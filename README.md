## Tech Stack

**Client:** React, Html/Css




## Run Locally

Clone the project

```bash
  git clone https://github.com/DucLT0809/my-react-test.git
```

Go to the project directory

```bash
  cd my-react-test
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
